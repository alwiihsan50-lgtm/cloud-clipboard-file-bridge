Quick Actions

Create two one-tap Home Screen shortcuts. A new key replaces the previous Quick Actions key.

CloudBridge Push

  1. Get Clipboard
  2. Get Contents of URL: POST, JSON body content = Clipboard
  3. Add header Authorization using the value above
  4. Show Notification: Sent
  5. Add to Home Screen

CloudBridge Pull

  1. Get Contents of URL: GET
  2. Add header Authorization using the value above
  3. If result has value, Copy to Clipboard
  4. Show Notification: Pulled
  5. Add to Home Screen
